# Planification de Trajectoire pour Robot Mobile avec SLPSO
# Réalisation-d-une-preuve-de-concept-du-SLPSO-pour-la-planification-de-trajectoires-de-robots-mobiles

## Description du projet
Ce projet présente une preuve de concept (PoC) pour la planification de trajectoire d’un robot mobile dans un environnement encombré d’obstacles statiques, en utilisant l’algorithme Self-Adaptive Learning Particle Swarm Optimization (SLPSO). L’objectif est d’évaluer les performances du SLPSO et de les comparer à celles du Particle Swarm Optimization (PSO) et de l’algorithme génétique (GA). 

## Méthodologie
- Implémentation de PSO, SLPSO et GA pour la planification de trajectoire.
- Génération d’environnements 2D avec obstacles statiques.
- Évaluation des trajectoires selon plusieurs critères : longueur, risque de collision, lissage.
- Comparaison des performances et analyse des résultats.

## Résultats
Les résultats montrent que le SLPSO produit des trajectoires plus courtes, plus lisses et plus sûres que le PSO et le GA, tout en assurant une meilleure stabilité et convergence. le resultat de comparaison sont disponible dans le rapport docs/PoC_SLPSO_Rapport.pdf.
